DECLARACIÓN DE ACCESIBILIDAD

Compromiso con la accesibilidad

En la nuestra enciclopedia médica, estamos comprometidos con garantizar accesibilidad a cualquiera, incluyendo a las personas con discapacidades.

Características de accesibilidad

  • Navegación mediante teclado: Nuestro sitio web es totalmente navegable usando solo el teclado. Usa la tecla Tab para moverte entre los enlaces y botones.
  • Textos alternativos para imágenes: Todas las imágenes importantes incluyen descripciones mediante atributos alt para que los lectores de pantalla puedan interpretarlas.
  • Identificación del idioma: Está indicado en cada apartado de la página qu el idioma principal es el español.
  • Etiquetado semántico: En toda la página se hace uso de etiquetado semántico, para facilitar su comprensión.
  • Enlaces descriptivos: Todos los enlaces de la página tienen descripciones claras que explican su destino y propósito, para mejorar la navegación.

Cómo mejorar la accesibilidad

Si tienes dificultades para acceder a cualquier parte de este sitio web o quieres hacernos llegar sugerencias para mejorar la accesibilidad, por favor, contáctanos a través de nuestro formulario de contacto.
